# Soda Bottle 3D Model Project

This is a test project created to demonstrate how to model a 3D soda bottle using a reference photo.

## Project Overview
- **Goal**: Create a realistic 3D model of a soda bottle based on a reference image.
- **Software Used**: Autodesk Fusion 360.
- **Output Format**: The final model was exported as an `.stl` file, optimized for importing into Bambu Labs slicers for 3D printing.

## Key Techniques Demonstrated
This project showcases several fundamental and intermediate modeling techniques in Fusion 360:
- **Line Sketches**: Creating the initial profile for the bottle.
- **Revolve/Extrude**: Generating the main body of the bottle from the sketch.
- **Shell**: Hollowing out the bottle to create a realistic thickness.
- **Fillets**: Adding smooth transitions and rounded edges to enhance realism and printability.
